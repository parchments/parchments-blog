# 图片预览 v-viewer

官网地址

Github地址

以指令形式使用使用步骤

1、安装 npm install v-viewer
2、main.js引入

//main.js
import Viewer from 'v-viewer' //导入图片预览
//引入Viewer基础样式
import 'viewerjs/dist/viewer.css'
//使用Viewer
Vue.use(Viewer);
